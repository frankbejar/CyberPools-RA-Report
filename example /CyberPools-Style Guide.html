Accessibility & Print

WCAG 2.1 AA Compliance

Color Contrast Requirements

  • Body text: Minimum 4.5:1 contrast ratio
  • Large text (18px+): Minimum 3:1 contrast ratio
  • Headings (bold): Minimum 3:1 contrast ratio
  • Never use color alone to convey information
  • Test with: WebAIM Contrast Checker

Typography Accessibility

  • Minimum font size: 12px (11px absolute minimum)
  • Line height: 1.5-1.6 for body text
  • Line length: Maximum 80 characters
  • Heading hierarchy: Proper H1 → H2 → H3 structure
  • Link text: Descriptive (not "click here")

Color Blindness Considerations

  • Use icons and text labels with colors
  • Avoid red/green combinations alone
  • Test with color blindness simulators
  • Use patterns in addition to colors where possible

Print Specifications

Print Setup

  • Paper size: US Letter (8.5" × 11")
  • Orientation: Portrait
  • Margins: 0.6in top/bottom, 0.7in left/right
  • Color mode: RGB for digital, CMYK for commercial
  • Resolution: 300 DPI minimum for images

Print Optimization

  • Page breaks: Avoid breaking cards, tables, findings
  • Headers/footers: Print on every page
  • Hide in print: Interactive controls, buttons, navigation
  • Test: Always preview before printing
  • PDF export: Use browser print-to-PDF function

Quality Checklist

Before Finalizing Reports

  • ☐ Client name correct throughout
  • ☐ Dates accurate and consistent
  • ☐ Color palette matches style guide
  • ☐ Typography consistent
  • ☐ Color contrast meets WCAG AA
  • ☐ Proper heading hierarchy
  • ☐ Tables properly formatted
  • ☐ All charts render correctly
  • ☐ Page breaks appropriate
  • ☐ Print preview looks correct
